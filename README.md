# Task 2: Customer Retention & Churn Analysis
## Future Interns - Data Science & Analytics Internship

### Project Overview
Analyzed customer data to identify churn patterns, key retention drivers, and customer lifetime trends to help improve business sustainability.

### Key Insights & Findings
* Churn by Tenure: Data indicates that newer customers have a significantly higher probability of churning compared to long-term clients.
* Contract Analysis: Month-to-month contracts are the primary source of churn; annual contracts provide much higher stability.
* Service Impact: Identified specific service types (like Fiber Optic) that correlate with higher churn rates, suggesting a need for service optimization.
* Payment Trends: Customers using electronic checks show higher churn levels than those on automated payment plans.
